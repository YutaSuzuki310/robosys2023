# Usage
「plus」という名前のpythonのプログラムファイルを作成してください。そして同様の名前のファイルのコードをgithubからコピー＆ペーストして保存して
$ seq "自然数" | ./plus
と入力すると指定した自然数までの階乗の値が表示されます。
