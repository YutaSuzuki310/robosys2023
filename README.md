# Usage
「plus」という名前のpythonのプログラムファイルを作成してください。そして同様の名前のファイルのコードをgithubからコピー＆ペーストして保存し
$ seq "自然数" | ./plus
と入力すると指定した自然数までの総和、総乗が表示されます。
