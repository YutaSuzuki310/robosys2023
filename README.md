# robosys2023_1
ロボットシステム学の課題１

# Usage
「plus」という名前のpythonのプログラムファイルを作成してください。

そして同様の名前のファイルのコードをgithubからコピー＆ペーストして保存し

$ seq "自然数" | ./plus

と入力すると指定した自然数までの総和、総乗の値が出力されます。

また少数は切り捨てなので"3.7"と入力した場合は"3.0"として計算されます。
